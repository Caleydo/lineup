=====================

LineUp is a Phovea application based on [LineUp.js](http://github.com/Caleydo/lineup.js)


![screenshot](media/screenshot.png?raw=true "Screenshot")

Installation
------------

```
git clone https://github.com/Caleydo/lineup.git
cd lineup
npm install
```

Testing
-------

```
npm test
```

Building
--------

```
npm run build
```
